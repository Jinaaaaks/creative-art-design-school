# Forma Studio — Creative Art & Design School

A fully responsive multi-page website for an imaginary creative arts and design school.

## Features

- Modern responsive layout
- Semantic HTML structure
- Shared design system (shared.css)
- Reusable JavaScript (shared.js)
- Interactive navigation and animations
- Filterable course listings
- Application and login pages

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
